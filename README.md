## 🚀 The Vision
**City Pulse** is a high-fidelity urban decision platform that resolves the **"Information Asymmetry"** crisis in modern cities. While traditional real estate platforms provide static, often misleading data (rent, proximity, amenities), City Pulse synthesizes thousands of real-time "Human Signals" from the NYC 311 Open Data API to reveal the **live operational truth** of every neighborhood.

## ⚠️ The Problem: "Zombie Data"
Newcomers, investors, and city officials currently make multi-million dollar decisions based on "Zombie Data"—information that is static and fails to capture the dynamic stress of urban life. 
*   **The Hidden Catch:** A building may look luxury on paper but suffer from chronic infrastructure decay.
*   **The Mobility Trap:** A neighborhood may be "near a subway" but have failing street-level transit health.

City Pulse eliminates these traps by turning chaos into **Strategic Intelligence**.

---

## 🧠 Core Features

### 1. Multi-Agent Intelligence Deck
The system employs three specialized AI agents to analyze urban signals borough-wise:
*   **🏢 Infrastructure Asset Risk Agent:** Analyzes building and utility signals to predict **Deferred Maintenance Costs** (CapEx Risk).
*   **📈 Transit-Driven ROI Agent:** Measures "Transit Health" to forecast **Property Value Retention**.
*   **🏪 Commercial Vitality Agent:** Uses sanitation and public health trends to predict **Local Business Stability**.

### 2. Intervention Simulator ("What-If" Engine)
A predictive modeling sandbox where users can test city-level policies before they are implemented. 
*   **Predictive Cascades:** Simulate an action (e.g., *Optimize Traffic Flow*) and watch the AI model the domino effect on public health and commercial vitality.
*   **Strategic ROI:** Quantify the "Recovery Value" of an intervention in real-time.

### 3. Tactical Visualization Layer
A geo-spatial map that clusters 311 signals into "Intensity Sectors," allowing for instant identification of urban stress clusters across all five boroughs.

## 🛠 Tech Stack
*   **Frontend:** React.js, Vite, Lucide-React (High-fidelity tactical UI).
*   **Backend:** FastAPI (Python), Uvicorn.
*   **AI Inference:** Groq SDK (Llama-3.1-8B-Instant) for sub-second strategic reporting.
*   **Data Pipeline:** Real-time Socrata API integration (NYC Open Data).
*   **Vector Engine:** ChromaDB (for historical pattern matching).

---

## 🎯 Best Use Cases
*   **High-Stakes Relocation:** Newcomers vetting an apartment building's "hidden" infrastructure history before signing a lease.
*   **Real Estate Investment:** Investors identifying "High-Recovery" blocks where small city interventions will drive massive value.
*   **City Management:** A unified command center for cross-agency collaboration between the DOT, DSNY, and DOB.

## ⚙️ Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Vritika0703/City-Pulse.git
   ```
2. Install Backend Dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Install Frontend Dependencies:
   ```bash
   cd frontend && npm install
   ```
4. Run the Platform:
   - Backend: `uvicorn app.main:app --reload`
   - Frontend: `npm run dev`
